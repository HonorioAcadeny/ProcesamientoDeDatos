## 🧮 ¿De qué se trata esta asignatura?

En el mundo de la tecnología y la informática, los datos y la información representan la base de cualquier sistema. Por ello, es importante saber manejarlos, gestionarlos, analizarlos y trabajar con ellos de manera eficiente, para así generar sistemas que nos permitan alcanzar los objetivos planteados.

Debemos entender que esta asignatura no se enfoca únicamente en la construcción de código óptimo, sino que abarca mucho más. Aquí no solo aprenderemos sobre estructuras de datos y programación, sino también sobre programación de calidad. Asimismo, veremos cómo organizar el trabajo de desarrollo de datos, optimizar procesos y crear productos de calidad.

Es por ello que debemos estar preparados para aprender a construir sistemas de manera integral: desde conversar con los usuarios y definir qué tipo de sistema es necesario para cumplir con las exigencias o resolver el problema planteado, hasta la organización ordenada de equipos de trabajo, la creación de prototipos y la verificación de calidad de los productos.
